# airflow-provision

A quick way to start Airflow with CeleryExecutor in Docker.

## How to run ?

```bash
sudo chmod +x start.sh
./start.sh
```

## Exec into airflow-worker

```bash
sudo chmod +x airflow.sh
./airflow.sh bash
```
